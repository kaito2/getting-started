## おまけ: taskipy

poetry では現状 `npm start` みたいな任意のスクリプトを登録することができなさそうなので([issue](https://github.com/python-poetry/poetry/issues/241#issuecomment-577142663))、 [taskipy](https://github.com/illBeRoy/taskipy) を使って同じようなことを実現してみた。

[GitHub - illBeRoy/taskipy: the complementary task runner for python](https://github.com/illBeRoy/taskipy)

# 実行

```
poetry run task server
```
